# 🧠 Agentic AI Learning Journey (30 Days)

A structured, hands-on, 30‑day journey into **Agentic AI**, designed for 1–2 hours of focused learning per day.  
This repository tracks daily progress, notes, experiments, and projects as I build the skills needed to create **real, production‑grade agentic systems**.

---

## 🚀 Goals of This Repository
- Build a strong foundation in LLMs, prompting, memory, and tool use  
- Learn how agent loops work (plan → act → observe → reflect)  
- Build single agents, then multi-agent systems  
- Use Spring AI + Java to build real agentic applications  
- Create reusable patterns, notes, and architecture diagrams  
- Track progress with daily checkboxes  
- Maintain long-term documentation for future projects
