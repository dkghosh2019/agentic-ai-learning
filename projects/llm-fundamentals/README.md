# 📘 Milestone: LLM Fundamentals

This milestone establishes the core understanding required to work with modern Large Language Models.  
It forms the foundation for all future agentic capabilities in this repository.

---

## 🎯 Objectives
- Understand how LLMs process text  
- Learn about tokens and context windows  
- Understand system, user, and assistant roles  
- Explore temperature, top‑p, and reasoning modes  
- Learn how LLMs structure outputs  
- Build intuition for prompt behavior  

---

## 📚 Topics Covered

### **1. What is an LLM**
- Transformer architecture (high level)
- Attention mechanism (conceptual)
- Why LLMs are probabilistic

### **2. Tokens & Context Windows**
- What tokens are  
- How tokenization works  
- Why context limits matter  

### **3. Prompt Structure**
- System prompts  
- User prompts  
- Assistant responses  
- Multi-turn context  

### **4. Model Parameters**
- Temperature  
- Top‑p  
- Max tokens  
- Reasoning modes  

### **5. Practical Exercises**
- Write simple prompts  
- Compare outputs with different temperatures  
- Explore structured output
